# Workflow de Desarrollo

Para iniciar el proyecto en modo desarrollo, sigue estos pasos:

1. **Backend**:
   ```bash
   cd backend
   npm run start:dev
   ```

2. **Frontend**:
   ```bash
   cd frontend
   npm run dev
   ```

> [!NOTE]
> Asegúrate de tener las variables de entorno configuradas y el motor de base de datos PostgreSQL activo.
