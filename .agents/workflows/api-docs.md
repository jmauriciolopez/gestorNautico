# API Reference

## Clientes
- `GET /clientes`: Listar todos los clientes.
- `POST /clientes`: Registrar un nuevo cliente.

## Embarcaciones
- `GET /embarcaciones`: Listar embarcaciones.
- `POST /embarcaciones`: Registrar embarcación.
